LINUX/KERNEL DEVELOPMENT

Linux Kernel Module Programmers Guide

Peter Jay Salzman, Michael Burian, Ori Pomerantz, Bob Mottram

Do you want to write a kernel module? Want to get into kernel programming but don't know where to start? Feel intimidated by millions of lines of impenetrable kernel code or frustrated by out of date documentation with examples that won't even compile? If so, then this book is for you.

The Linux Kernel Module Programmers Guide introduces you to the basics of how to write a module, from "hello world" to scheduling and elementary driver development. This isn't a deep dive into Linux's subsystems but does give you a starting point from which you can continue on your adventures. Each section provides runnable examples so that you can get hands-on experience of how the kernel works.

What's Inside

  • Setup of the development toolchain
  • C code examples with explanations of how they work
  • Guidance on where to get further help

This book assumes readers understand the basics of C programming.

Bob Mottram is a free software developer with a focus on federated, autonomous and self-hosted server systems. He maintains a blog at blog.libreserver.org.
